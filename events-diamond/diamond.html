Clash of Coders

It is a programming event to test the fundamentals of the most popular programming languages of all participants. The rounds will start with the basic concepts and slowly buildup to challenge their programming and logic application skills. This event will help developers to test and re-evaluate their basics and beginners to solidify their concepts. The languages permitted in this event are C, C++, java and Python.

Registration fees: ₹ 50.

● No choice in programming languages will be given in first round.
● Coding rounds will be either in C or C++.
● Do not generate the entire pattern using only ‘printf’ or ‘cout’ statements. Use your logic.
● Internet access is prohibited for referring code solution.
● Generated pattern should be exactly the same as given in the problem statement.
● If anyone is found violating these rules, they will be immediately disqualified from the event.

Round 1: Quiz
● All participants will have MCQs from languages like C, C++, Java and Python.
● Each participant will be given 60 randomly selected MCQ questions to solve within 30 minutes.
● On the basis of how many questions are answered correctly, the participant’s performance will be evaluated.
● After grading all students, the top scorers will be selected for the next round.

Round 2: Pattern Generation
● All participants will have MCQs from languages like C, C++, Java and Python.
● Each participant will be given 60 randomly selected MCQ questions to solve within 30 minutes.
● On the basis of how many questions are answered correctly, the participant’s performance will be evaluated.
● After grading all students, the top scorers will be selected for the next round.

Round 3: Advanced programming
● Each participant will be given a problem statement and they will have to satisfy all necessary objectives given using programming language specified.
● The first participant who successfully implements all modules of the problem statement correctly will be declared the winner.

Brain Chase

This event is to provide participants a platform to check their both Non-technical and technical skills which will be team event(Group of two). Event in the beginning will start with the Non-Technical round to boost confidence among participants but also will be tricky to get through! Next round onwards technical knowledge will be tested. This event will help participants to test their programming skills and how well they cooperate with each another.

100 INR(Rs.50 each)

● Internet access will be provided only for the first round(i.e Googler).
● Use of pen/pencil and paper is allowed.
● No prompting of answers or any kind of discussion is allowed, if found may get the participant disqualify after certain warnings.
● Use of programming languages C and C++ will be allowed for round 2(i.e Code relay).
● Output shall not be generated directly using ‘printf’ and ‘cout’ statements.
● Single entries will be paired up with another single entry randomly right from round one.

Round 1: Googler
● 15questions will be provided to the participants.
● A time limit of 15 min will be given.
● Each question will carry 1 mark.
● Participants can google answer to every question.
● Qualification for the next round will be decided on marks allotted to questions and time.
● Number of participants qualifying for next round will be decided from the number of enrolments for the event.

Round 2: Code Relay
● One participant solves a program for 20min while other waits in waiting room and after 20min they switch their places.
● Programs(approx. 5-7) will be provided to the first participant which he is supposed to choose randomly from chits.
● Time limit of 20min will be given for first participant.
● After 15min the second participant will be called in to solve the remaining program.
● A window of 2minutes will be given to them to discuss the program after 20min of first participant.
● Number of participants qualifying for next round will be selected in this round depending on the required output and time.

Round 3: Blind Coding
● The participants will be given chits of programs from they can select any 1.
● The programs need to be solved in either C or C++.
● The monitors of the participants will be turned off when the participants are ready.
● They will be provided with pen and paper for rough work.
● Maximum time for this event given will be One hour.
● After completion of programs by the participants, the programs will be evaluated on the basis of correct output and least numbers of errors using which the winners will be decided.

Mock Placement

● It gives a platform to experience the placement event.
● It will help you to find out what is the placement procedure and where do you stand.
● Candidates will be participating individually.
● Entry fees will be 100 per team.

● No internet access for any rounds.
● Use of calculator is allowed
● If anyone found cheating, he will be immediately disqualified.
● Final decisions will rest with the organizers.

Round 1: Aptitude test
● It is a MCQ test in which participants will be given 30 questions(Based on aptitude topics) ● The participants will be given 30 minutes to solve the maximum possible questions. ● The participants who will get the maximum score will be selected for the next round.

Round 2: Group Discussion
● One topic will be given to group of 3-5 candidates. ● The participants will be given 3 minutes to prepare. ● According to performance top 2 candidates from each group will be selected for the next round.

Round 3: Technical Interview
● The interviewer will conduct the technical round ● It will be based on C,C++ & Java language (Or specified by sponsor) ● According to performance candidates will be selected for the next round.

Round 4: HR Interview
It will be final round and based on the performance candidates will be selected for the internship or letter of appreciation.

TV Trivia

It is a non-technical event which has got Several Western TV Shows named
1) FRIENDS
2) GAME OF THRONES
3)BREAKING BAD
4) SUITS
5) BROKLYN 99
6)SACRED GAMES
7)MIRZAPUR
8)HISTORICAL MARATHI SERIAL

Entry Fees: RS 50/-

● Internet access for only the third round.
● If anyone found cheating, he will be immediately disqualified.
● Final decisions will rest with the organizers.

Round 1: Quiz
● A multiple-choice question bank of around 30-40 questions will be formed
and the difficulty level is minimum in this round.
● A Criteria will be set for selecting and eliminating participants.
● Time constraint is included.
● The timer will start as soon as the participants take the quiz.
● Selected participants will go to the next round.

Round 2: Can you guess the dialogue

● Two contest will face each other in this round. Both contestants will stand across each other & will have 5 cups on the table.
● Each contestant will try to land a ball into cup across them, on successful try they will have to assemble a dialogue from TV series mentioned on the cup.
● They will get 15-20 sec to do.
● First person to get all five dialogues correct will be qualified for next round.

Round 3: Rapid Fire
● The difficulty level will be tough in this round
● A series of questions will be set and ‘OPTIONS WILL NOT BE GIVEN’.
● Questions can be in the form of series scenes/dialogues completion.
● There is no reverse questioning, once the question is proceeded the participant cannot ask for the previous question to the respective volunteer.
● Time constraint is included.
● Participants with the accurate answers will go to the next round.
